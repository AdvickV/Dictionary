# Dictionary
 A Desktop App that gives the Definition, Pronunciation, Grammar Type and Examples of the word entered. 
# Libraries
1. Pillow
```
$ pip install Pillow
```
2. Pyttsx3
```
$ pip install pyttsx3
```
3. Requests
```
$ pip install requests
```
# Usage
- Type a word in the Entry.
- Click "Search Word" Button.
